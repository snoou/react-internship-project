#  React Internship Project


## نصب و اجرا

برای اجرای پروژه روی سیستم خود، مراحل زیر را انجام دهید:

```bash
# 1. کلون کردن پروژه
git clone https://github.com/YourUsername/react-internship-project.git

# 2. وارد پوشه پروژه شوید
cd react-internship-project

# 3. نصب وابستگی‌ها
npm install

# 4. اجرای پروژه
npm start
